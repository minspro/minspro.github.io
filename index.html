案例

基于python+aiohttp的WEB代理服务器

基于 Python 和 aiohttp 构建的异步 HTTP/HTTPS 代理服务器，支持请求修改转发与响应修改和回传，适用于高并发网络场景。

基于 libevent 的 TCP/TLS 加解密代理服务器

在现代网络通信中，TLS 加密已成为标准配置。然而，在某些调试、审计或测试场景下，我们需要对加密流量进行解密和分析。本文介绍了一种基于 libevent 和 OpenSSL 实现的 TCP/TLS 加解密代理服务器方案，能够在不解耦底层协议的前提下，实现对客户端 TLS 流量的透明解密与转发。

基于 Suricata 的数据库流量审计系统设计与实践

在企业级网络安全架构中，数据库作为核心资产常常成为攻击者的主要目标。为了实现对数据库访问行为的全面监控和异常检测，本文提出了一种基于 **Suricata** 的 **非侵入式数据库流量审计方案**。通过旁路抓包 + Suricata 规则匹配的方式，实现对数据库协议（如 MySQL、MSSQL、PostgreSQL）的实时解析与行为分析，无需修改业务逻辑或部署探针。
